# Projeto Galaga (Terminal)

Jogo estilo Galaga feito em Python para rodar no terminal usando `curses`.

## 🔐 Pre-requisitos

- Python 3.8+ instalado
- Windows: `curses` pode exigir ambiente compatível (ex: Windows Terminal). Em Linux/macOS funciona nativamente.

## Estrutura do projeto

- `Galaga2.py`: jogo principal
- `placar_galaga.txt`: criado automaticamente para salvar o placar

## Instalação do projeto

Nao requerida.

## Executando a aplicação

```bash
python Galaga2.py
```

## Controles

- `A`: move para a esquerda
- `D`: move para a direita
- `S`: atira

## Regras basicas

- Desvie de inimigos e obstaculos
- Obstaculos nao podem ser destruidos
- Cada inimigo destruido soma pontos
- O jogo termina quando as vidas acabam
